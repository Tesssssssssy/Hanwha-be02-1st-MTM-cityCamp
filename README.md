캠핑을 즐길 수 있도록 여러 루프탑 등의 장소들을 중개 및 제공하는 서비스이다.

# ✒️ 프로젝트 배경
일상에서 벗어나고 싶지만 멀리 가기 힘든 사람들을 위해

"도심 속 캠핑"이라는 주제로 프로젝트가 시작되었다.

<br>
공실인 원룸, 상가 등을 보유한 사람들이 장기, 단기 임대를 주기 어려운 상황,

모텔, 파티룸 등에는 가기에 부담스럽고, 또 멀리 가기 힘든 사람들의 상황 등에 적합한 중개 플랫폼이다.
